<section id="visao-geral">
    <h2>Visão Geral</h2>
    <p>Este projeto é uma landing page destinada a uma loja de roupas online. A página foi projetada para ser atraente e fácil de navegar, proporcionando uma experiência de usuário agradável. Além disso, a landing page destaca produtos em destaque, promoções e possui seções para depoimentos de clientes, informações sobre a loja e um formulário de inscrição para newsletter.</p>
</section>

<section id="funcionalidades">
    <h2>Funcionalidades</h2>
    <ul>
        <li><strong>Design Responsivo:</strong> A página se ajusta automaticamente a diferentes tamanhos de tela, proporcionando uma boa experiência em dispositivos móveis e desktops.</li>
        <li><strong>Carrossel de Imagens:</strong> Destaque para produtos em promoção ou lançamentos.</li>
        <li><strong>Seção de Produtos:</strong> Exibição de produtos populares com imagens, descrições e preços.</li>
        <li><strong>Depoimentos:</strong> Área dedicada para feedbacks de clientes satisfeitos.</li>
        <li><strong>Formulário de Newsletter:</strong> Captura de emails para campanhas de marketing.</li>
        <li><strong>Rodapé Informativo:</strong> Informações de contato, links para redes sociais e políticas da loja.</li>
    </ul>
</section>
